Get Google Cloud Translate API client through one of the following:
> composer require google/cloud-translate 
> https://php-download.com/package/google/cloud-translate

Create your Google Cloud project, get your authentication JSON, and put it in the auth subfolder here. Learn more at:
> https://cloud.google.com/apis/docs/getting-started

Configure your project so that the user specified in your authentication file could have enough privileges to run the translations. It does not happen automatically, it has to be set. Learn more from Google official doc at:
> https://cloud.google.com/iam/docs/granting-changing-revoking-access
